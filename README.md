### Installing the project
Start by cloning the project

    $ git clone https://github.com/divinityMaker/pokemon-cards-nextjs
    
    
Enter the project folder

    $ cd pokemon-cards-nextjs

And install it

    $ yarn install

##### Running
In the project's folder

    $ yarn run dev
